## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Swagger Docs

  To access the swagger you can click the http://localhost:3000 and it will open the swagger docs.
You can use the swagger to make a sample request to each endpoints.

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
